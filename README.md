# Epitech Project - 42sh

### Description
The goal of the **42sh** project was to develop in C a shell inspired by the TCSH. This project follow up on two previous solo projects :
- the **minishell1** project consisted to create the structure of the shell able to ask a command to the user and execute them
- the **minishell2** project consisted to add output and input redirections (simple and double) (for instance : `ls > file.txt`,
`grep 'hello' < file.txt`, `ls | grep 'name' >> output.txt`, ...)
The **42sh** is the outcome of the 'Shell Programming' module consisting to add as features as possible to the **minishell2** of one of the group members
(the project had to be done by groups of 4 or 5 peoples).
